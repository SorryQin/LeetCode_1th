## 509. 斐波那契数

这道题目难度不大，属于是一到动态规划的入门问题，而且已经明确给出了dp递推数组的递推公式，直接写代码即可

实现代码如下：

```c++
class Solution {
public:
    int fib(int n) {
        if (n <= 1)  return n;
        int num[2];
        num[0] = 0;
        num[1] = 1;
        for (int i = 2; i <= n; i++) {
			int sum = num[0] + num[1];
            num[0] = num[1];
            num[1] = sum;
        }
        return num[1];
    }
};
```
