## 62.不同路径

这道题目的思路还是很清晰的，只要确定初始的值以及递推的关系式，就可以很清楚地根据思路来写代码

<img src="https://github.com/SorryQin/LeetCode_1th/blob/main/data/3.17.1.png" width="50%">

实现代码如下：

```c++
class Solution {
public:
    int uniquePaths(int m, int n) {
        int nums[m][n];
        for (int i = 0; i < m; i++) {
            // line
            nums[i][0] = 1;
        }
        for (int j = 0; j < n; j++) {
            // row
            nums[0][j] = 1;
        }
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                nums[i][j] = nums[i - 1][j] + nums[i][j - 1];
            }
        }
        return nums[m - 1][n - 1];
    }
};
```
